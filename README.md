# CREACT

CREACT is a Python practice repository.

## Included examples
- `pod1.py` - variable swapping example
- `calculator.py` - basic calculator utility functions
- `app.py` - simple e-commerce backend starter using Flask
- `models.py` - in-memory data models for products, cart, and orders

## Run the e-commerce backend starter

```bash
pip install -r requirements.txt
python app.py
```

## Example API endpoints
- `GET /products`
- `POST /products`
- `GET /cart`
- `POST /cart`
- `POST /orders`
- `GET /orders`
